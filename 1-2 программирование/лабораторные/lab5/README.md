# Лабораторная работа 5

Учебная консольная Java-программа для работы с коллекцией объектов `Vehicle`.

Проект реализует команды для добавления, обновления, удаления, фильтрации, сортировки, сохранения коллекции и выполнения команд из скрипта.

## Основные возможности

- загрузка коллекции из JSON-подобного файла;
- хранение коллекции `Vehicle`;
- добавление и обновление элементов;
- удаление элементов по `id` и по условию;
- фильтрация по полям;
- подсчёт элементов по количеству колёс;
- выполнение команд из файла через `execute_script`;
- сохранение коллекции обратно в файл;
- автодополнение команд по `Tab` в интерактивном режиме;
- Javadoc-комментарии на русском языке.

## Структура проекта

```text
src/
├── cmd/       # запуск программы и команды
├── data/      # классы данных: Vehicle, Coordinates, FuelType
├── parser/    # парсер коллекции из файла
└── storage/   # хранение и операции над коллекцией

scripts/       # примеры скриптов и данных
resources/     # ресурсы проекта
lib/           # внешние библиотеки, включая JLine для автодополнения
```

## Команды программы

- `help` — вывести справку по доступным командам;
- `info` — вывести информацию о коллекции;
- `show` — показать все элементы коллекции;
- `add` — добавить новый элемент;
- `update id` — обновить элемент по `id`;
- `remove_by_id id` — удалить элемент по `id`;
- `clear` — очистить коллекцию;
- `save` — сохранить коллекцию в файл;
- `execute_script file_name` — выполнить команды из файла;
- `exit` — завершить программу без сохранения;
- `head` — вывести первый элемент коллекции;
- `add_if_max` — добавить элемент, если он больше максимального;
- `remove_greater` — удалить элементы, превышающие заданный;
- `count_by_number_of_wheels numberOfWheels` — посчитать элементы с указанным количеством колёс;
- `filter_by_fuel_type fuelType` — вывести элементы с указанным типом топлива;
- `filter_starts_with_name name` — вывести элементы, имя которых начинается с заданной строки.

## Запуск

Для запуска нужно указать переменную окружения `FILE_NAME`, в которой хранится путь к файлу с коллекцией.

Пример для macOS/Linux:

```bash
export FILE_NAME=scripts/data.json
javac -cp lib/jline-3.30.0.jar -d out src/**/*.java
java -cp out:lib/jline-3.30.0.jar cmd.Main
```

В интерактивном режиме можно нажимать `Tab` для автодополнения команд. Например,
`sho` дополняется до `show`, а для `he` выводятся варианты `head` и `help`.

Пример запуска скрипта внутри программы:

```text
execute_script scripts/FirstFile
```

## Примечание

Это учебный проект, поэтому код сделан простым и понятным. Для обработки
нажатия `Tab` используется библиотека JLine.
